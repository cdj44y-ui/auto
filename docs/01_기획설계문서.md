## 1. 기획서 / 요구사항 정의서

### 1.1 프로젝트 개요

근태관리시스템(AES)은 기업의 인사·급여 관리를 통합적으로 지원하는 엔터프라이즈급 웹 애플리케이션입니다. 출퇴근 관리, 휴가 신청, 급여 계산, 전자 근로계약서 등 HR 업무 전반을 디지털화하여 업무 효율성을 극대화하고 법적 준수를 자동화합니다.

### 1.2 프로젝트 목표

| 목표 영역 | 세부 목표 | 성공 지표 |
|----------|---------|----------|
| **업무 효율화** | 수작업 HR 업무의 80% 이상 자동화 | 월간 HR 업무 시간 50% 감소 |
| **법적 준수** | 근로기준법 자동 반영 | 주 52시간 초과 근무 0건 |
| **데이터 보안** | 민감 정보 보호 및 감사 추적 | 보안 감사 로그 100% 기록 |
| **사용자 경험** | 직관적인 UI/UX 제공 | 사용자 만족도 4.0/5.0 이상 |

### 1.3 기능 요구사항

#### 1.3.1 핵심 기능 (Core Features)

**출퇴근 관리**
- 실시간 출퇴근 기록 및 상태 표시
- IP 기반 접근 제어
- 지각/조퇴/결근 자동 판별
- 실근무 시간 자동 계산

**직원 관리**
- 직원 정보 등록/수정/삭제 (개별 및 일괄)
- 엑셀 파일 일괄 업로드
- 부서/직급별 필터링
- 직원 상태 관리 (재직/휴직/퇴직)

**급여 관리**
- 급여 대장 관리
- 급여 명세서 자동 생성
- 이메일 발송 (개별/일괄)
- 4대 보험 자동 계산

**휴가 관리**
- 연차 관리 대장
- 휴가 신청/승인 워크플로우
- 팀 휴가 캘린더
- 잔여 연차 자동 계산

#### 1.3.2 고급 기능 (Advanced Features)

| 기능 | 설명 | 상태 |
|-----|------|-----|
| 유연근무제 | 시차출퇴근, 재택근무, 선택근무제 지원 | ✅ 완료 |
| 교대근무 스케줄러 | 3교대, 2교대 등 교대 근무 일정 관리 | ✅ 완료 |

| 연장근무 리포트 | 연장/야간/휴일 근무 통계 | ✅ 완료 |

#### 1.3.3 보안 기능 (Security Features)

- **RBAC (Role-Based Access Control):** admin, hr, finance, user 4단계 권한 체계
- **보안 감사 로그:** 모든 중요 작업 기록 및 추적
- **민감 정보 마스킹:** 주민등록번호, 계좌번호 등 자동 마스킹
- **IP 접근 관리:** 허용된 IP에서만 출퇴근 기록 가능
